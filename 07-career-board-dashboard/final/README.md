# CareerBoard Dashboard

I built CareerBoard as my complete MERN practice project. I can register, manage my profile, track job applications, filter records, paginate results, and review application statistics from one dashboard.

## What I practiced

- I built an authenticated Express and MongoDB API.
- I managed client state and asynchronous requests with Redux Toolkit.
- I created protected React routes and reusable form components.
- I displayed application trends with charts.
- I implemented search, filters, pagination, and status summaries.

## How I run it

I copy `.env.example` to `.env`, provide my MongoDB and JWT values, and install the server dependencies. I install the React dependencies inside `client`, build the client with `npm run build`, and start the server from this folder with `npm start`.
