# FocusFlow Task Manager

I built FocusFlow as my first full-stack CRUD exercise. I use an Express API and MongoDB to create, read, update, complete, and delete tasks from a responsive browser interface.

## What I practiced

- I designed REST endpoints for task CRUD operations.
- I connected Express to MongoDB with Mongoose.
- I handled asynchronous errors with reusable middleware.
- I connected a plain JavaScript frontend to my API with Axios.

## How I run it

I copy `.env.example` to `.env`, provide my MongoDB connection string, run `npm install`, and start the project with `npm start`.
