# Product Image Uploader

I built this exercise to upload product images through Express and store them with Cloudinary. I combine the uploaded image URL with product data and display the saved catalog in the browser.

## What I practiced

- I handled multipart form uploads with Express.
- I connected a Node.js application to Cloudinary.
- I validated uploaded files and product data.
- I rendered uploaded products in a responsive gallery.

## How I run it

I copy `.env.example` to `.env`, provide my MongoDB and Cloudinary test credentials, run `npm install`, and start the project with `npm start`.
