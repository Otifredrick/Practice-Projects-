# Product Catalog API

I built this API to practice turning URL query parameters into useful MongoDB queries. I can filter products, compare numeric values, select fields, sort results, and paginate the catalog.

## What I practiced

- I implemented dynamic filtering and numeric operators.
- I added sorting, field selection, and pagination.
- I created a population script for sample product data.
- I added a simple landing page that links to the JSON endpoint.

## How I run it

I copy `.env.example` to `.env`, provide my MongoDB connection string, run `npm install`, optionally run `node populate.js`, and start the API with `npm start`.
